## 2235. Add Two Integers

Given two integers `num1` and `num2`, return the **sum** of the two integers.

### Code

```python
class Solution(object):
    def sum(self, num1, num2):
        """
        :type num1: int
        :type num2: int
        :rtype: int
        """
        return num1+num2
        
```

### Complexity

- Time complexity: **O(1)**
- Space complexity: **O(1)**
